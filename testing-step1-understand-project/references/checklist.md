# 勘察/测试覆盖检查清单（逐项打勾·泛化）

> 配合 SKILL.md 第 5/7 节使用。🟡 处为"按项目替换的可变项"，非默认值。

## A. 环境与部署
- [ ] 唯一权威入口已锁定，排除同名残留(旧容器/进程/库)
- [ ] 健康检查 / 首页 / 关键接口连通
- [ ] 反向代理路由（/api、/static、/uploads、特殊代理如 /live）已梳理
- [ ] 编排与配置（compose/env）：DB 指向、密钥、**特性开关**、体积/超时/Token 时效
- [ ] 安全头：X-Frame-Options / nosniff / HSTS / CSP / CORS / Permissions-Policy
- [ ] 特性开关对"代码逻辑 vs 线上实际"的影响已评估（如种子/迁移仅某开关下执行）

## B. 认证与会话
- [ ] 登录成功/失败响应（失败是否统一文案，防用户枚举）
- [ ] 空账号/空密码/超长/特殊字符（前端校验 + 直接打 API 绕过）
- [ ] Token 机制：算法、时效、刷新、登出、过期后行为
- [ ] 失败锁定/限流/验证码 是否存在
- [ ] 客户端存储（localStorage/cookie/IndexedDB）存了什么、是否敏感
- [ ] 多端/并发会话、记住账号

## C. 权限与越权（每个受保护资源都过）
- [ ] 权限矩阵：无token / 各低权角色 / 高权 三态状态码
- [ ] 前端路由守卫 vs 后端接口鉴权 是否对等
- [ ] 越权读：低权访问他人资源；越权写：匿名/低权 增删改受保护数据（**写需授权，用可逆往返+自清理**）
- [ ] 数据隔离：列表是否按 owner 过滤
- [ ] 角色字段是否可越权篡改（无枚举约束的 role）
- [ ] 批量接口的鉴权与越权

## D. 表单与输入边界（每个输入框都过）
- [ ] 必填留空（逐字段）
- [ ] 超长 / 最小长度 / 空白串归一化
- [ ] 特殊字符 / 注入样本 / emoji / 中文 / 前后空格
- [ ] 数值：min-1 / min / max / max+1 / 小数 / 负数 / 非数字
- [ ] 格式正则：手机号/身份证/邮箱/日期/UUID 的合法与非法样本
- [ ] 唯一约束：重复值
- [ ] 默认值是否生效、是否随场景变化
- [ ] **前后端校验差异**（同字段两端规则是否一致；一端无校验另一端有→后果是什么）
- [ ] maxLength/showCount/输入掩码

## E. 状态机与业务流
- [ ] 每个状态的允许操作（按状态枚举逐一）
- [ ] 合法流转全链路覆盖；非法流转（错误来源状态触发动作）→ 错误码/文案
- [ ] 幂等（重复提交/重复创建）
- [ ] 级联删除/关联约束（删除带子数据）；多步创建的**原子性/回滚**（中途失败是否残留脏数据）
- [ ] 并发操作同一资源

## F. 列表/表格/分页
- [ ] 分页大小、翻页、总数一致性
- [ ] 搜索（每个可搜字段）、筛选组合、重置、排序
- [ ] 空态文案、加载态文案、错误态
- [ ] 统计卡数字与列表/后端口径一致（**核对聚合公式**）

## G. 文件/媒体/第三方 🟡 条件触发
> 🟡 **仅当项目确有文件上传/导出/媒体/第三方依赖时本节才适用；无此类的项目（如纯表单 CRUD）本节整节跳过，不计漏测**（与已条件化的 I 节、`non-standard-forms.md` 看齐）。命中其中某一类（如确有上传）则该项必测。
- [ ] 上传：类型限制、大小上限（前端/网关/后端各一道）、超限文案
- [ ] 导出：格式、文件名规则、中文、内容正确（含图）、**导出机制（前端生成 vs 后端生成）**
- [ ] 图片/媒体显示链路（直链/代理/CDN/CORS；**先判断有无换过存储/CDN——换过则必查**新旧路径与旧数据可达性、迁移一致性）
- [ ] 实时流/音视频：连接/断开/重连/降级提示；实测是否真在播（读 video.readyState 等）
- [ ] AI/外部服务：成功/失败/超时/弱网 错误码；隐私与配额；返回值是否引用了已删除资源

## H. 富交互/画布/快捷键 🟡 条件触发
> 🟡 **命中即必测、未命中整节跳过**：控件**确实支持**该交互（拖拽/悬停/滑块/右键/滚轮/canvas 绘制/快捷键）时即必测；**无此类交互的项目（标准表单/列表页）本节整节跳过，不计漏测**（与 I 节、`non-standard-forms.md` 看齐）。富 UI 项目命中时严格度一丝不减。
- [ ] 画布/标注工具逐个、颜色/线宽、撤销/删除/清空
- [ ] 快捷键（呼出/确认/取消）
- [ ] 离开拦截/未保存提示
- [ ] 弹窗/抽屉的确认与取消文案

## I. 显示与文案
- [ ] 硬编码常量是否被误当数据（型号/时间/版本号等）
- [ ] 未映射枚举的兜底渲染（显示原始英文/空白）
- [ ] 脏数据/异常值（负值、超范围、历史脏字段）的展示
- [ ] 时区/日期格式/本地化
- [ ] **运行期文本替换**（DOM 文本被改写→源码词≠显示词）

## J. 安全（授权范围内）
- [ ] 无鉴权高危端点清单
- [ ] CORS/CSP 配置
- [ ] 密钥/凭证是否明文暴露（配置/前端）
- [ ] 代理类接口的 SSRF/滥用面
- [ ] 错误信息是否泄露内部细节
